# Linux Disk Monitor

A Flutter application for Linux that monitors attached disks in real-time using a C++ backend.
It displays detailed information such as disk name, type, total size, used/available space, and mount points.

**Features:**
- **Real-time Monitoring:** Streams updates every 2 seconds via EventChannels.
- **Visual Usage Indicators:** Color-coded bars show disk usage status.
- **Dual Modes:** Supports both static data fetching and continuous live streaming.
- **Modern UI:** Clean interface built with Material Design 3.
